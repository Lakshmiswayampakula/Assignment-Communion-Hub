# Communion Hub

## "Bringing People of All Faiths Together Through Events & Community Support"

Communion Hub is a web application dedicated to creating a welcoming space where faith and fellowship flourish. We empower individuals to connect, share, and grow spiritually through meaningful events, volunteer opportunities, and enriching discussions.

## 🌟 Features

- **Community Events**: Browse and join faith-based events in your area
- **Discussion Forums**: Engage in meaningful spiritual conversations
- **ChatBot**: Get instant answers to common questions about our community
- **Join Community**: Simple registration process to become part of our family
- **Categories**: Explore different faith traditions and practices
- **Responsive Design**: Seamless experience across all devices

## 🛠️ Technologies

- **Frontend**: React.js, JavaScript
- **Styling**: Tailwind CSS
- **Development**: Node.js
- **Version Control**: Git, GitHub
- **Deployment**: Netlify

## 🚀 Getting Started

### Prerequisites

- Node.js (version-latest)
- npm (v-latest)
- vs code

### Installation

1. Clone the repository
```bash
git clone https://github.com/your-username/communion-hub.git
cd communion-hub
```

2. Install dependencies
```bash
npm install
```

3. Start the development server
```bash
npm start
```

4. Open your browser and visit `http://localhost:3000`

## 🌐 Deployment

The project is deployed using Netlify.

### Deployment Steps

1. Build the project
```bash
npm run build
```

2. Deploy to Netlify
   - Connect your GitHub repository to Netlify
   - Set the build command to `npm run build`
   - Set the publish directory to `build`
